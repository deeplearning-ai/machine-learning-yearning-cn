## 合作改善翻译质量

译者水平有限，如有翻译不当之处，恳请读者帮助斧正，途径为：

1. 在项目的 Issues 区创建新的讨论内容；（推荐）
2. 在网页点击 “改进当前页面”，修改后发起 Pull Request；
3. [发送邮件](mailto:xiaowei@deeplearning.ai) 进行反馈，主题带有 “MLY 翻译”。
